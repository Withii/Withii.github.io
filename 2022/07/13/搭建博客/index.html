docker配置nginx

首先安装nginx镜像
docker pull nginx
安装好可以选择测试(也可以跳过):测试命令两行
docker run –name nginx-test -p 80:80 -d n
docker start nginx-test

将nginx关键目录映射到本机

在主机创建映射目录
创建www⽬录
mkdir -p /server/nginx/html
创建⽇志⽬录
mkdir -p /server/nginx/logs
创建配置⽬录
mkdir -p /server/nginx/conf
拷⻉容器内 Nginx 默认配置⽂件到本地当前⽬录下的 conf ⽬录，容器 ID 可以查看 docker ps 命令输⼊中的第⼀列：
拷⻉配置⽂件
docker cp 容器id:/etc/nginx/nginx.conf ~/server/ngi
删除原来用于拷贝配置文件的nginx容器
docker stop 容器id
dokcer rm 容器
映射容器⽬录
/docker run -d -p 80:80 –name nginx \ -v
/server/nginx/html:/usr/share/nginx/html \ -v
/server/nginx/conf/nginx.conf:/etc/nginx/nginx.conf \ -v
/server/nginx/logs:/var/log/nginx \ –privileged=true nginx:1.17.8

添加反向代理

server{
    listen 80;
    server_name ikeguang.com www.ikeguang.com;
    location / {
    proxy_set_header Host $host;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_pass http://宿主机ip:8080;
    }

}

重启 nginx 容器

docker restart 容器ID
